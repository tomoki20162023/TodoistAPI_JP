制限事項

現在、POSTリクエストに対し、HTTPリクエスト本体の1MiB制限がある。このルールの例外はすべての アップロード エンドポイントで、100MiBの制限となっている。

Upload endpoints are for POST requests pointed at paths such as /upload and /uploads/add.

処理のタイムアウト

各エンドポイントには処理のタイムアウトがあり、実行されるアクションの種類に応じて変わります。

Type Limit
アップロード 5 minutes
Standard Request 15 seconds

Headers

Total size of HTTP headers cannot exceed 65 KiB.

毎分のリクエスト数

REST APIを使うときは、各ユーザが１分当たり50リクエストまでとなっています。
