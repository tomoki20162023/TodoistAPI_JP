ラベル

ラベルオブジェクトの例

{
    "id": 1234,
    "name": "Movies",
    "color": 30,
    "order": 10,
    "favorite": false
}

{
    "id": 1234,
    "name": "Movies",
    "color": 30,
    "order": 10,
    "favorite": false
}

属性

属性 説明
id Integer Label id.
name String Label name.
color Integer Color id. It’s a value between 30 and 49, refer to Colors for more info.
order Integer Number used by clients to sort list of labels.
favorite Boolean Whether the label is a favorite (a true or false value).

全てのラベルを取得する

全てのラベルを取得する

$ curl "https://api.todoist.com/rest/v1/labels" \
    -H "Authorization: Bearer $token"

[
    {
        "id": 1234,
        "name": "Movies",
        "color": 30,
        "order": 10,
        "favorite": false
    }
]

import requests
requests.get("https://api.todoist.com/rest/v1/labels", headers={"Authorization": "Bearer %s" % your_token}).json()

[
    {
        "id": 1234,
        "name": "Movies",
        "color": 30,
        "order": 10,
        "favorite": false
    }
]

全ユーザのラベルを含むJSONエンコードされた配列を返す。

新しいラベルを作成する

新しいラベルを作成する

$ curl "https://api.todoist.com/rest/v1/labels" \
    -X POST \
    --data '{"name": "Movies"}' \
    -H "Content-Type: application/json" \
    -H "X-Request-Id: $(uuidgen)" \
    -H "Authorization: Bearer $token"

{
    "id": 1234,
    "name": "Movies",
    "order": 10,
    "favorite": true
}

import requests
requests.post(
    "https://api.todoist.com/rest/v1/labels",
    data=json.dumps({
        "name": "Movies"
    }),
    headers={
        "Content-Type": "application/json",
        "X-Request-Id": str(uuid.uuid4()),
        "Authorization": "Bearer %s" % your_token
    }).json()


{
    "id": 1234,
    "name": "Movies",
    "order": 10,
    "favorite": true
}

新しいラベルを作成し、JSONとしてオブジェクトを返す。

JSON本体のパラメータ

パラメータ 要求 説明
name String Yes Name of the label.
order Integer No Label order.
color Integer No Color id (See colors).
favorite Boolean No Whether the label is a favorite (a true or false value).

ラベルを更新する

ラベルを更新する

$ curl "https://api.todoist.com/rest/v1/labels/1234" \
    -X POST \
    --data '{"name": "Movies"}' \
    -H "Content-Type: application/json" \
    -H "X-Request-Id: $(uuidgen)" \
    -H "Authorization: Bearer $token"

import requests
requests.post(
    "https://api.todoist.com/rest/v1/labels/1234",
    data=json.dumps({
        "name": "Movies"
    }),
    headers={
        "Content-Type": "application/json",
        "X-Request-Id": str(uuid.uuid4()),
        "Authorization": "Bearer %s" % your_token
    })

Updates a label and returns an empty body with a HTTP status code 204.

JSON本体のパラメータ

パラメータ 要求 説明
name String No New name of the label.
order Integer No Number that is used by clients to sort list of labels.
color Integer No Color id (See colors).
favorite Boolean No Whether the label is a favorite (a true or false value).

ラベルを削除する

ラベルを削除する

$ curl -X DELETE "https://api.todoist.com/rest/v1/labels/1234" \
    -H "Authorization: Bearer $token"

import requests
requests.delete("https://api.todoist.com/rest/v1/labels/1234", headers={"Authorization": "Bearer %s" % your_token})

Deletes a label and returns an empty body with a HTTP status code 204.
