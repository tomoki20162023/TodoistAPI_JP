認証

認証されたリクエストとレスポンスの例

$ curl -X GET \
  https://api.todoist.com/rest/v1/projects \
  -H "Authorization: Bearer $token"

[
    {
        "id": 1234,
        "name": "Inbox",
        "comment_count": 0
    }
]

import requests  
requests.get("https://api.todoist.com/rest/v1/projects", headers={"Authorization": "Bearer %s" % your_token}).json()

[{u'comment_count': 0, u'id': 1234, u'name': u'Inbox'}]

APIエンドポイントは URLエンコードされた 値としてGETの引数を受理します。全てのエンドポイント requires an 認証ヘッダー with 適切な Bearer $token that holds the Todoist personal API token もしくは OAuth API token with 全権限範囲.

Note that we’re using $token on all of our curl examples, so you can define 一時的な環境変数 containing 自身の token and easily コピペ curl コマンド into 端末.
