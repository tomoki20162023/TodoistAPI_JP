概要

これはTodoist Sync APIの非公式ドキュメントです。（訳者注：意図的に非公式としています。GitHubに載せている間はこのままになると思います）私たちの公式なAPIの機能的な参考文書を提供しています。各APIエンドポイント、パラメータそしてサンプルの詳細な説明が付いています。

要約

始めにの節で、みなさんにTodoist APIを示します。できるだけ簡単な方法で、例を使いながら、多くのユーザが達成したいであろう共通のタスクを基にしています。

この節を読んだあと、あなたは認証の節も読み進めるべきです。 私たちのサーバへ認証を行う最善の方法を、すべて詳細に学ぶことができます。

最も大事な節はSyncです。そして、次にAPIがどう動いているか説明しているところを読み進めるのがいいでしょう。

その他の節は別のTodoistのオブジェクトとエンドポイントの参考文書となっていますので、あなたが必要な通りに読み進めて下さい。

Python library

私たちは現在、生産活動が簡単になるように、Doistで公式のPythonライブラリを提供しています。

pipからtodoistのpythonライブラリをインストールできます:

$ pip install todoist-python

GithubのリポジトリからPythonライブラリのソースコードを見ることもできます。

Pythonライブラリの文書をオンラインで読むこともできます。

最後に、インストールが必要な方のためにPyPIパッケージも用意しています。
