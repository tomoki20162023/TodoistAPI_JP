Properties

Property Description
id Integer Section ID
project_id Integer ID of the project section belongs to
order Integer Section position among other sections from the same project
name String Section name

Get all sections

Get all sections

$ curl -s -H "Authorization: Bearer $token" \
    https://api.todoist.com/rest/v1/sections

[
    {
        "id": 1234,
        "project_id": 5678,
        "order": 1,
        "name": "Agenda"
    }
]

import requests
requests.get("https://api.todoist.com/rest/v1/sections", headers={"Authorization": "Bearer %s" % your_token}).json()

[
    {
        "id": 1234,
        "project_id": 5678,
        "order": 1,
        "name": "Agenda"
    }
]

Returns JSON array of all sections. Successful response has 200 OK status and application/json Content-Type.

Create a new section

Create a new section

$ curl -s -X POST --data '{"project_id":1234, "name":"Morning routine"}' \
    -H "Content-Type: application/json" \
    -H "Authorization: Bearer $token" \
    https://api.todoist.com/rest/v1/sections

{
    "id": 6789,
    "project_id": 1234,
    "order": 1,
    "name": "Morning routine"
}

import requests, json
requests.post(
    "https://api.todoist.com/rest/v1/sections",
    data=json.dumps({
        "name": "Morning routine"
    }),
    headers={
        "Content-Type": "application/json",
        "Authorization": "Bearer %s" % your_token
    }).json()

{
    "id": 6789,
    "project_id": 1234,
    "order": 1,
    "name": "Morning routine"
}

Creates a new section and returns it as a JSON object. Successful response has 200 OK status and application/json Content-Type.

Parameters

Parameter Required Description
name String Yes Section name
project_id Integer Yes Project ID this section should belong to
order Integer No Order among other sections in a project

Update a section

Update a section

$ curl -s -X POST --data '{"name":"Morning routine"}' \
    -H "Content-Type: application/json" \
    -H "Authorization: Bearer $token" \
    https://api.todoist.com/rest/v1/sections/1234

import requests, json
requests.post(
    "https://api.todoist.com/rest/v1/sections/1234",
    data=json.dumps({"name": "Morning routine"}),
    headers={
        "Content-Type": "application/json",
        "Authorization": "Bearer %s" % your_token
    }).json()

Updates section for the given ID. Successful response has 204 No Content status and an empty body.

Parameters

Parameter Required Description
name String Yes Section name
